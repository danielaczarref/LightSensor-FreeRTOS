# Light Sensor Project with FreeRTOS Arduino Kernel

## Objective

To light up a red LED when the ambient light lower than 40 from analog read.

## Components

- Arduino UNO;
- USB;
- Protoboard;
- Jumpers;
- LDR Luminosity Sensor 5mm;
- 300 Ohm Resistor.
